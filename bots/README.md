# Bot Clients

This directory contains example bot implementations that connect to the Master Bot system.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Update MASTER_BOT_URL in trading_bot.py with your server IP

3. Get your BOT_ID from the Master Bot registration system

4. Run the bot:
   ```bash
   python trading_bot.py
   ```
